## Decisions made during planning

The spec is the contract. A few clarifications resolved during planning that the engineer should NOT re-derive:

1. **`chrome.*` not `browser.*`.** The spec uses `browser.*` symbolically. Existing entrypoints (`background.ts`, `offscreen/main.ts`) call `chrome.*` directly. `chrome.alarms.*` and `chrome.runtime.connect` return Promises in MV3 on all three browsers. Use `chrome.*` to match local convention. Type via the existing `@types/chrome` dep.
2. **`alarms` permission is already in the manifest.** [`apps/extension/wxt.config.ts:22`](../../../apps/extension/wxt.config.ts#L22) already lists `'alarms'` — no manifest change needed.
3. **Tests colocated with source.** Per-package tests live next to source (e.g. `packages/llm/src/router.test.ts`). The cross-cutting integration test lives in `tests/gate/alarms.test.ts` (sibling of `tests/gate/wired.test.ts`).
4. **No dev-only RPC route needed for the integration test.** The spec contemplated one for triggering `ensureAlarms()` from a test. Cleaner: `background.ts` calls `void ensureAlarms()` at top-level on SW bootstrap, so loading the extension in Playwright creates the alarms automatically. The test asserts via `chrome.alarms.getAll()` evaluated in the newtab page context. Drops one task.
5. **Playwright config lives at the repo root** (`playwright.config.ts`) and the test file at `tests/gate/alarms.test.ts` so the `gate:alarms` workflow can `pnpm playwright test tests/gate/alarms.test.ts` symmetrically with how `gate:offline` runs `pnpm vitest run tests/gate/offline.test.ts`.
6. **Dependency injection for scheduler / handlers test seams.** `ensureAlarms()` accepts an optional `AlarmsApi` parameter (defaults to `chrome.alarms`); `registerAlarmHandlers()` accepts an optional `AlarmEvents` parameter (defaults to `chrome.alarms.onAlarm`). Production calls pass nothing → use the global. Tests pass mocks. Avoids global module mocking inside a non-WXT package boundary.

export async function withHeavyDocAlive<T>(work: () => Promise<T>): Promise<T> {
  await ensureHeavyDoc();
  const port = chrome.runtime.connect({ name: 'heavy-doc-keepalive' });
  try {
    return await work();
  } finally {
    port.disconnect();
  }

import { BRIEFING_HOUR, REFLECTION_HOUR } from './defaults';

export interface DesiredAlarm {
  name: string;
  when: number;
}

function nextOccurrenceAtHour(hour: number): number {
  const now = new Date();
  const candidate = new Date(now);
  candidate.setHours(hour, 0, 0, 0);
  if (candidate.getTime() <= now.getTime()) {
    candidate.setDate(candidate.getDate() + 1);
  }
  return candidate.getTime();
}

export function computeDesired(): DesiredAlarm[] {
  return [
    { name: 'morning-brief', when: nextOccurrenceAtHour(BRIEFING_HOUR) },
    { name: 'eod-reflection', when: nextOccurrenceAtHour(REFLECTION_HOUR) },
  ];
}

```ts
export interface AlarmsApi {
  getAll(): Promise<Array<{ name: string; scheduledTime: number }>>;
  create(name: string, info: { when: number }): Promise<void>;
  clear(name: string): Promise<boolean>;
}

const RESCHEDULE_TOLERANCE_MS = 60_000;

function defaultApi(): AlarmsApi {
  return chrome.alarms as unknown as AlarmsApi;
}

export async function ensureAlarms(api: AlarmsApi = defaultApi()): Promise<void> {
  const existing = await api.getAll();
  const desired = computeDesired();

  for (const d of desired) {
    const e = existing.find((a) => a.name === d.name);
    if (!e || Math.abs(e.scheduledTime - d.when) > RESCHEDULE_TOLERANCE_MS) {
      await api.clear(d.name);
      await api.create(d.name, { when: d.when });
    }
  }

  for (const e of existing) {
    if (!desired.find((d) => d.name === e.name)) {
      await api.clear(e.name);
    }
  }
}
```
